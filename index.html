Simple Calendar

View the Code on GitHub

Simple Calendar was my final project when I attended the Iron Yard. It is a single-page application that makes use of an Angular JS directive to render data from a Ruby on Rails JSON API. User authentication was accomplished using Devise on the server-side and Angular-Devise for the client-side, with necessary validations on both fronts. All requests are asynchronous, thus never requiring the user to refresh the page.

Technologies: AngularJS, Ruby on Rails, Devise, Angular UI Calendar, Zurb Foundation
